# 기본 화면


## 리눅스 시스템 명령어 (pdf p.50)
- 화면에서 현재활동 > 터미널에서 입력
reboot : 재부팅 명령어
clear: 화면 전체 삭제 명령어
pwd: 현재 자신의 경로 확인
#: 주석이라 실행되지 않음. 쉘스크립트를 만들어서 실행시킬 때도 무시하는 줄이 된다.
cd: 경로를 이동할 때 사용하는 명령어. 그냥 cd를 하면 user 디렉토리로 온다.
ls: 현재 경로의 폴더 목록 보기
