# Linux: Day 2
## 기본적인 명령어 - 계속
+ v1 editor 사용  
  + vi file명  
  + i - 데이터 입력
  + esc - 데이터가 작성이 다되면 esc키를 누른다  
  + :를 누른 후 wq(write, quit)해서 파일을 빠져나간다.
  + :wq
+ q!: 강제로 파일을 나올 때
+ cat: 파일내용 확인  
+ 세부 옵션 확인 - 명령어 --help
