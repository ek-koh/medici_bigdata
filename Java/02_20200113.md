## pbv vs. pbr
- 기본타입은 value를 주고 value를 받는다.
```
pass by value
- assign by value
- immutable
```
```java
int a = 10;
int b = a; // a -> b
c = a + 20;
System.out.println(a); // 10
// a에게 어떤 작업을 하더라도 값이기 때문에 변화지 않는다.
```

- 참조타입은 reference를 주고 reference를 받는다.
```
pass by reference
- assign by reference
- mutable
```
```java
int[] c = {1,2,3,4,5};
int[] d = new int[5];
d = c;
c[2] = 100;
System.out.println(d[2]); //
```
